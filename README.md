# Review Tag | Frontend

Review Tag는 영화·콘텐츠 리뷰 플랫폼입니다. 리뷰, 별점, 가격 평가, 퀴즈, 포인트 기능이 한 서비스 안에서 이어지도록 만든 팀 프로젝트입니다.

프론트에서는 포인트 화면 쪽을 맡았습니다. 출석, 일일 퀘스트, 상점, 인벤토리, 아이콘, 이력, 관리자 화면이 각각 따로 보이지 않고 한 흐름처럼 이어지게 만드는 쪽에 더 집중했습니다.

---

## 프로젝트 소개

- 영화·콘텐츠 리뷰와 가격 평가를 함께 다루는 서비스
- 사용자 활동을 포인트 적립과 소비 경험으로 연결한 구조
- 로그인 사용자 기준으로 출석, 퀘스트, 상점, 인벤토리, 아이콘, 이력, 랭킹 기능 제공

---

## 담당 역할

- 포인트 메인 화면과 하위 탭 구성
- 출석 체크 / 출석 캘린더 연결
- 일일 퀘스트, 룰렛, 포인트 선물하기 연동
- 상점, 위시리스트, 인벤토리, 아이콘, 이력 화면 구현
- 관리자 포인트, 상품, 자산 화면 연동
- 프로젝트 공통 axios / Jotai 구조를 기준으로 포인트 화면 갱신 흐름 정리

---

## 사용 기술

- React, Vite
- React Router
- Axios
- Jotai
- Git, GitHub

---

## 구현하면서 신경 쓴 점

- 출석, 구매, 아이템 사용 뒤 다른 탭까지 값이 늦지 않게 반영되도록 정리했습니다.
- 상점에서는 상품 목록만이 아니라 보유 상태와 위시 상태를 함께 보여주도록 맞췄습니다.
- 프로젝트에 이미 있던 공통 인증 구조 위에서 포인트 화면 쪽 요청과 갱신 기준을 통일했습니다.
- 관리자 화면도 사용자 화면과 완전히 다른 흐름으로 두지 않고 같은 기준 안에서 연결했습니다.
